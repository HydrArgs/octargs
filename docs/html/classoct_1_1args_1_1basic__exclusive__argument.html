Detailed Description

template<typename char_T, typename values_storage_T = internal::null_values_storage>
class oct::args::basic_exclusive_argument< char_T, values_storage_T >

Exclusive argument.

Exclusive arguments are switch arguments (arguments without value) that could be only used alone when specified in input. For example –version or –help.
